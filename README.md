## JavaScript Loops & Array Methods

This section explains different ways to iterate through arrays and objects, along with commonly used array methods like `map`, `filter`, and `reduce`.

---

## Looping Through an Array

JavaScript provides multiple ways to loop through an array:

- **For Loop**: A traditional approach where you control the index manually. Useful when you need access to the position of elements.
- **forEach()**: A built-in method that executes a function for each element in the array. Cleaner and more readable for simple iterations.
- **for...of Loop**: Iterates directly over the values of an array, making it easy to work with elements without worrying about indexes.
- **for...in Loop**: Iterates over the indexes (keys) of the array. It is generally used for objects but can also be applied to arrays.

---

## Looping Through an Object

Objects in JavaScript can be iterated using the `for...in` loop.  
This loop allows you to access each key in the object and retrieve its corresponding value. It is useful when working with structured data like user profiles or records.

---

## Using `map()`

The `map()` method is used to transform elements in an array.

- It creates a **new array**
- Applies a function to each element
- Does **not modify** the original array

This is commonly used when you want to update or process data without changing the original dataset.

---

## Using `filter()`

The `filter()` method is used to select specific elements from an array based on a condition.

- Returns a **new array**
- Includes only elements that satisfy the condition
- Does not modify the original array

Useful for tasks like finding values greater than a certain number or extracting specific data.

---

## Using `reduce()`

The `reduce()` method is used to combine all elements of an array into a single value.

- Iterates through the array
- Accumulates a result (such as sum, product, etc.)
- Returns a single output

It is often used for calculations like totals, averages, or combining data into one result.
